# Installation Guide for GT-IconScaler

## System Requirements
- **Operating System**: Windows 10 or later, macOS 10.14 or later, Linux.
- **Memory**: Minimum 4GB RAM (8GB recommended).
- **Storage**: At least 500MB of free disk space.
- **Dependencies**: 
  - Node.js (version 14 or later)
  - npm (Node Package Manager)

## Installation Instructions

### Step 1: Clone the Repository
Open your terminal and clone the repository using the following command:
```
git clone https://github.com/SalehGNUTUX/GT-IconScaler.git
```

### Step 2: Navigate to the Directory
Change to the directory of the cloned repository:
```
cd GT-IconScaler
```

### Step 3: Install Dependencies
Before running the application, make sure to install the necessary dependencies:
```
npm install
```

### Step 4: Run the Application
After the dependencies have been installed, you can start the application by running:
```
npm start
```

## Troubleshooting

- **Issue**: Application fails to start.
  - **Solution**: Ensure Node.js and npm are installed correctly. You can check the version using `node -v` and `npm -v`. 

- **Issue**: Dependencies not installing.
  - **Solution**: Check your internet connection and try running `npm install` again. If the problem persists, consider clearing the npm cache with `npm cache clean --force`.

- **Issue**: Application crashes during usage.
  - **Solution**: Ensure your system meets the system requirements and all dependencies are installed. Check the console for error messages that might indicate what's wrong.
